SimpleX Chrome Extension
Tech Stack: JavaScript, HTML, CSS

Simplex is a simple new tab page chrome extension - that's it!

Each time you visit a new tab, you can view an inspirational quote pulled from favqs.com API, the current date, and time in your set region.

SimpleX is focused on keeping your digital life simple and minimal.

See the GitHub Repo here: https://github.com/KeshavChawla/SimpleX.
